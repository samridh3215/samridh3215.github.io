## Overview

This project implements a biometric payment system that enables students to make on-campus purchases using fingerprint authentication. The system is inspired by the tab-based payment model used at BITS Pilani Hyderabad campus, where students can purchase food, beverages, and other items on credit, with balances cleared during semester fee payments.

## Key Features

- Secure biometric authentication using fingerprint recognition
- Tab-based payment system for deferred settlement
- Real-time transaction processing and tracking
- Seamless integration with campus commerce ecosystem

## Technology Stack

- **Backend API**: Flask framework for RESTful services
- **Hardware**: Raspberry Pi with integrated fingerprint sensor
- **Database**: MongoDB for transaction and user data management
- **Authentication**: Biometric fingerprint verification

## Applications

This system enhances the campus payment experience by eliminating the need for physical cards or cash, providing a secure and convenient payment method for students.
